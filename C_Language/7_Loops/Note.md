# 7강. 반복문
* 일정한 조건을 만족할 때까지 특정 코드를 반복해서 실행하는 제어문

## 1. for문
* 반복 횟수가 정해져 있을 때 사용.
  ```
  for (초기화; 조건; 증감) {
    // 반복 실행할 코드
  }
  ```

## 2. while문
* 조건이 참인 동안 계속 반복
  ```
  while (조건) {
    // 반복 실행할 코드
  }

## 3. do-while문
* 반복 전에 조건을 먼저 확인하지 않고, 코드 블록을 최소 한 번 실행한 후 조건을 확인
  ```
  do {
    // 반복 실행할 코드
  } while (조건);
  ```
